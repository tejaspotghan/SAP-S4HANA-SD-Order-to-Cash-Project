# SAP S/4HANA SD Order-to-Cash Project

## Project Overview

This portfolio project demonstrates an end-to-end Order-to-Cash process in SAP S/4HANA Sales for a fictional construction chemicals company.

The project shows how a dealer order for 100 bags of Type 2 Tile Adhesive moves through the complete sales process.

## Business Scenario

**Company:** BuildBond Chemicals Pvt. Ltd.  
**Industry:** Construction Chemicals  
**Customer:** Balaji Tiles & Sanitary  
**Customer Type:** Dealer  
**Product:** Type 2 Tile Adhesive  
**Order Quantity:** 100 Bags

## Complete Process Flow

**Customer Order → Sales Order → Outbound Delivery → Picking → Post Goods Issue → Billing**
